각 버튼의 역할
🌐 웹사이트 헤더
↺ 새로고침Cloudflare KV에서 최신 포트폴리오 데이터를 다시 불러와 화면 갱신. 현재가는 크롤링하지 않음.
+ 종목 추가새 종목을 Sheets에 추가하고 KV 캐시도 즉시 갱신.
📈 추이분석 탭
↻ 스냅샷 새로 읽기KV의 snapshot_data 캐시를 삭제하고 Sheets에서 최신 스냅샷을 재로드. Sheets 직접 수정 후 사용.
💾 스냅샷 저장현재 포트폴리오 상태를 Sheets 스냅샷 시트에 즉시 저장. 매일 오후 4~5시 자동저장 외 수동 저장.
💰 배당 탭
↻ 새로 읽기배당내역 시트를 다시 조회. Sheets에서 배당 데이터 직접 수정 후 사용.
+ 배당 입력새 배당 내역을 Sheets에 추가.
💸 실현손익 탭
↻ 새로 읽기실현수익 시트를 다시 조회. Sheets에서 직접 수정한 경우 사용.
+ 매도 입력매도 기록 입력. 기본 동작으로 투자현황 수량이 자동 차감됨.
✏️ 수정기존 매도 기록 수정. 수량 변경 시 투자현황도 자동 재조정.
🗑️ 삭제매도 기록 삭제 = 매도 취소. 투자현황 수량이 자동 원복됨.
⚙️ GAS 메뉴 (Google Sheets → 📈 주식관리)
⚡ 지금 즉시 현재가 갱신 환율+현재가 크롤링 → Sheets 업데이트 → KV 캐시 갱신. 약 30초 소요.
Sheets에서 종목/계좌 수정 후 반드시 실행.
📸 추이 스냅샷 지금 저장 현재 포트폴리오를 스냅샷 시트에 즉시 저장. 웹의 💾 스냅샷 저장과 동일.
🎨 투자현황 계좌별 색상 투자현황 시트 행을 계좌별로 색상 구분. 선택사항.
🔃 KRX 종목코드 갱신 KRX에서 전체 종목 코드를 즉시 갱신. 매일 오전 6시 자동 실행.
⚙️ 트리거 재설정 10분 자동갱신 트리거가 사라졌을 때 재등록.
🔗 웹앱 URL / ☁️ Cloudflare 설정 최초 1회 설정 후 변경할 일 없음.
⏰ 자동 실행 주기: 현재가 갱신 10분마다 · KRX 종목코드 갱신 매일 오전 6시 · 스냅샷 자동저장 매일 오후 4~5시
💸 실현손익 탭 개요
주식 매도 시 실현한 수익/손실을 기록·집계하는 탭입니다.
매도 기록 저장 시 투자현황 시트의 수량이 자동 차감되며, 삭제 시 자동 복원됩니다.

📈 양수 = 팔고 나서 더 올랐다 (아쉬운 상황) / 📉 음수 = 팔고 나서 내려갔다 (잘 판 상황)
오늘 등락: 해당 종목의 오늘 등락률 (보유 중인 경우만 표시)
⑤ 수정 · 삭제 시 투자현황 자동 처리
✏️ 수정: 수량 변경 시 변경분(delta)만큼 투자현황 재조정. 완전매도→부분매도로 변경 시 삭제된 행이 자동 복원됨
🗑️ 삭제: 매도 자체를 취소. 투자현황 수량이 원래대로 복원됨. 완전매도로 행이 삭제됐던 경우도 자동 복원
